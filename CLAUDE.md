# CLAUDE.md — このリポジトリで Claude Code に頼むとき

このリポジトリは **Claude Code 環境再現キット**（`claude-code-env01`）です。

## セットアップを頼まれたら（最重要）

ユーザーが「このリポジトリを読んで環境をセットアップして」「Claude Code の初期設定をして」等と言ったら、
**必ず `SETUP.md` を読み、その手順に厳密に従って**オンボーディングを実行してください。

要点（詳細は `SETUP.md`）:

- ユーザーの**今の環境を先に調べ**、このキットと**双方向に比較**する（リポジトリは強制標準ではなく選択肢のメニュー）。
- **差分だけ**を1項目ずつ「○○（やさしい説明）を導入しますか？ YES/NO」で確認する。
- **既存設定は壊さない**（必ずバックアップ→マージ。置換しない）。
- ユーザー側の設定の方が優れている / 本人の用途・環境に合わない項目は**無理に入れず、その都度判断**する。
- **非エンジニアにも分かる言葉**で説明しながら進める。日本語で。

## 手動で入れたい人へ

人間が自分でインストールする場合は `README.md` と `setup.sh`（コンポーネント選択式インストーラ。`--yes`/`--skip`/`--minimal` 対応）を参照。

## このリポジトリ自体を編集するとき

`claude/` 配下はユーザー環境に配置される雛形、`SETUP.md` はエージェント向け手順、`setup.sh` は人間向けインストーラ。秘密情報・個人情報・個人パス（`/Users/<名前>` 等）は**絶対に含めない**（`$HOME` で表現する）。
